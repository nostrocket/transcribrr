## Divergence view design (RPT-04/05)

### Granularity / what counts as divergence

| Option | Description | Selected |
|--------|-------------|----------|
| Sentence, normalized | Align at sentence/segment level; normalize (lowercase + strip punctuation + collapse whitespace) before comparing, display original text | ✓ |
